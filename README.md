# AI-Powered Clinical Trial Assistant

This project is a Streamlit-based application that uses Retrieval-Augmented Generation (RAG) to analyze clinical trial PDFs and answer user queries.

## Features
- Multi-PDF upload
- Semantic search using embeddings
- LLM-based question answering (Ollama + Gemma)

## Tech Stack
- Streamlit
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Ollama (Gemma)

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
